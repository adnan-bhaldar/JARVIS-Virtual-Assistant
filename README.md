### Prerequisites

### Run Locally

To run **JARVIS** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/adnan-bhaldar/JARVIS-Virtual-Assistant.git
```

Windows:

```bash
git clone https://github.com/adnan-bhaldar/JARVIS-Virtual-Assistant.git
```
